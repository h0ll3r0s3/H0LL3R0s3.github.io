2、Windows11

Windows 11 和Windows10 大同小异，不过替换以后会提示0x2350 因为在环境变量中的设置路径和运行的cmd路径不符合导致的。尝试添加用户也会出错，然后搜索了很久发现另外一种方式，直接利用Netplwiz工具。(**盲猜:**不出意外直接可以利用Netplwiz直接替换即可无需cmd修改密码。)

科普：Netplwiz.exe是一个Windows工具，用于在Windows10/11上管理用户账户,也是一个Windows运行命令，可以让用户轻松地删除在Windows登陆时输入密码的需要。也可以使用此命令轻松禁用密码登录或者禁用Windows自动登录、添加用户、更改权限等操作。位于C:\Windows\System32\ 。

直接在上文中替换好的cmd下输入Netplwiz即可弹出用户账户控制面板。即可进行操作(仅限于本地操作)。
